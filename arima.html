Predicting Price - ARIMA Model

  1. Predicting future trends with ARIMA

We tried leveraging ARIMA, a popular machine learning model for predicting time series data by trying to find predictable patterns that repeat over a calendar year. However, as seen in the graph it:

  • Does not predict any seasonal trend as prediction is a straight line.
  • Overpredicts test data at the start from 2018-2020, however, it starts to underpredict from mid 2020, similar to the regression model.
