Predicting Price - Regression Model

A multi linear regression model (MLR) was made to compare the various factors using 3 data of factors affecting resale HDB flats from 2018 to 2020 and was used to test on 2021 data. The MLR model can help users be more informed when purchasing houses for investment as they can estimate the cost of the house they want to purchase based on different factors.

Factors:

  1. Floor Area per square meter
  2. Remaining lease
  3. Distance to closest hospital (km)
  4. Distance to closest hawker centre (km)
  5. Distance to closest MRT distance (km)
  6. Distance to closest park (km)
  7. Distance to closest clinic (km)
  8. Distance to closest mall (km)
  9. Distance to closest market (km)
  10. Distance to closest school (km)
  11. Different towns (values 0 or 1)

After running the regression for the first time, distance to closest school was found to be insignificant as the p value was 0.4179. Since it is greater than 0.05, we removed the factor and ran the model again.

Testing of the model obtained a r2 score of 0.70 (2dp) and RMSE of 88,916.09. This means that 0.7 of the variance of the resale price can be explained by the independent variables. The RMSE shows the avg difference between actual and predicted value. (e.g. if predicted price is $500,00 the expected value is expected to be between 411,083.91 and 588,916.09)
